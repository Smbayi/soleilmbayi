# Soleil Mbayi — Portfolio

Portfolio personnel de **Soleil Mbayi** (*Le Tout Puissant Soleil*).

## Stack

- Next.js 15 · TypeScript · Tailwind CSS 4
- Framer Motion · GSAP · React Three Fiber
- Lucide Icons · Dark mode

## Démarrage

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000).

## Pages

- `/` — Portfolio immersif
- `/univers` — Mode « Découvrir mon univers »
- `/philosophie` — Le Code, la Culture et l'Humanité
- `/cv` — CV imprimable / PDF

## Langues

Français (défaut) et Anglais via le sélecteur FR / EN dans la navigation.
